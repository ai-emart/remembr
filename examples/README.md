# Examples

This repository keeps runnable reference architectures in `examples/`.

## Flagship Example

- [LangGraph Multi-Agent Memory Loop](./langgraph-multi-agent/README.md)
  A research -> writing -> review workflow where all three agents share one Remembr session and retrieve prior feedback instead of bloating the live prompt.

## Adapter Mini-Demos

Smaller adapter-specific demos also live under `adapters/*/examples/`, but the LangGraph multi-agent example is the best place to start if you want a full, reviewable pattern to fork.
